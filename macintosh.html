Macintosh

Macintosh is the line of Apple's PCs and portable computers. This line was started in 1984 and currently contains 3 types of computers: MacBook (laptop), iMac (all-in-one PC), and Mac (system unit)

I don't want to create the second Wikipedia so if you want to see all of the Macintosh computers launched by Apple, check this Wikipedia article.
